
# FihCast

FihCast is a website made for helping people determine if the day is good or not good to go out fishing based on certain factors like: 

- Weather temperature,
- Wind speed,
- Estimated water temperature,
- Chance of rain.

Based on these factors the website gives you a rating from 1 to 10 on this rating scheme:

- Wind: 0-3 points,
- Weather temperature: 0-3 points,
- Rain chance: 0-2 points,
- Pressure / cloud cover: 0-2 points.

# Usage

The website is made available for phones and computers alike with a responsive interface making it easy to use on smaller devices like: smartphones

The website is reachable with the url listed on the about section of this github repository or with the link below:

https://CBalazs1.github.io/FihCast
